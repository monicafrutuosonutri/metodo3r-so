# Roteiro — maldicao-vigilancia

## Cabeçalho

- **Formato:** Carrossel Tema Forte
- **Tema:** A Maldição da Vigilância: como ela se instala sem você perceber
- **Pilar:** A Maldição
- **Tese:** A Maldição da Vigilância não é fraqueza — é um sistema instalado. E sistemas se desinstalam com método.
- **Crença/visão construída:** O problema nunca foi você. Foi o sistema que a dieta construiu dentro de você.
- **Total de slides:** 8

**DIREÇÃO VISUAL:**
Dois blocos visuais separados por linha fina ou espaço maior. O título "A Maldição da Vigilância:" em peso maior. O segundo bloco ("Não é X / Não é X / É Y") com ritmo visual de lista — cada linha em sua própria linha.

### SLIDE 4 — COMO SE INSTALA

```
Como ela se instala:
```

*Lista:*
```
→  Você começa uma dieta

→  Aprende a monitorar tudo

→  Seu cérebro entende:
   comida = perigo

→  Cria um alerta automático

→  A dieta acaba.
   O alerta, não.
```

*Abaixo, separado:*
```
Não foi fraqueza.
Foi adaptação.
```

**DIREÇÃO VISUAL:**
As setas (→) criam ritmo visual de processo. Cada item em linha própria com espaçamento. A última linha — "A dieta acaba. O alerta, não." — pode ganhar destaque visual (negrito, cor diferente ou tamanho levemente maior). "Não foi fraqueza. Foi adaptação." separado por espaço ou linha fina — é o fechamento do raciocínio, não parte da lista.

## Anotações para Produção (Mack)

**QUANTIDADE DE SLIDES:** 8

**PALETA SUGERIDA:**
- S1 (capa): fundo escuro (preto ou bordô) com texto branco
- S2-S5, S7-S8: fundo claro (off-white ou branco) com texto escuro
- S6 (contra-intuitivo): fundo levemente diferente — marcar visualmente a virada
- Cor de destaque: mesma cor da identidade visual da Mônica, aplicada nos números (S5), nas etapas 3R (S7) e no punch do S6

**TIPOGRAFIA:**
- Headline/Punch: sans-serif bold ou black, tamanho grande
- Corpo: mesma família, peso regular, tamanho legível em mobile
- Hierarquia visual clara — texto principal é sempre maior que o secundário

**PROPORÇÃO:**
- 1:1 (quadrado) ou 4:5 (levemente vertical) — ambos funcionam no feed
- Testar legibilidade em tela de celular antes de publicar

**ELEMENTOS GRÁFICOS:**
- Mínimos — a copy é o visual
- Setas (→) no S4 como elemento gráfico simples
- Linha fina de separação no S3 (entre definição e "Não é X / Não é X")
- Nenhum ícone, ilustração ou foto — pode competir com o texto

**LEGIBILIDADE MOBILE:**
- Fonte mínima de 18-20pt equivalente em tela de celular
- Nunca mais de 8-10 linhas de texto por slide
