# Roteiro — contar-caloria-dependencia

## Cabeçalho

- **Formato:** Reel Lo-fi (fala direta, autoridade)
- **Tema:** Quando você não consegue comer sem contar, isso não é controle. É dependência.
- **Pilar:** A Perspectiva
- **Tese:** O problema não é a ferramenta. É quando você não consegue mais comer sem ela.
- **Crença/visão construída:** Controle real é quando você escolhe usar a ferramenta. Dependência é quando você não consegue não usar.
- **Duração estimada:** 50-58 segundos
- **Versão atual:** v2 — 17/06/2026

---

## Hook — 3 primeiros segundos

**VERBAL:**
> "Quando você não consegue comer sem contar, isso não é controle. É dependência."

**VISUAL:**
Mônica olhando diretamente para a câmera. Expressão firme e calma — a de quem vai dizer algo importante. Ambiente doméstico simples.

**TEXTUAL (texto na tela):**
"quando você não consegue comer sem contar"
*(pausa — depois aparece:)*
"isso não é controle."

**GATILHOS:** Disrupção + Polêmica

---

## Roteiro Completo — v2

*(versão corrida, pronta para gravar)*

---

"Quando você não consegue comer sem contar, isso não é controle. É dependência.

E antes de continuar — eu não tô falando que app de caloria é errado. A ferramenta, em si, pode ter o seu lugar.

Mas existe uma diferença entre usar a ferramenta e depender dela.

Quem usa pode parar quando quer. Quem depende — entra em pânico quando o app não carrega.

Você já ficou ansiosa porque não sabia a caloria de um prato? Não conseguiu pedir no restaurante sem pesquisar antes?

Isso não é o app funcionando pra você. É você funcionando pro app.

Controle de verdade é quando você consegue comer sem o app. Não quando você não consegue comer sem ele.

Isso já aconteceu com você? Me conta aqui."

## Análise Técnica

**LOOPINGS:**

| Looping | Onde abre | Onde fecha |
|---------|-----------|-----------|
| L1 — "mas qual é a diferença?" | 0:07 — "mas existe uma diferença" | 0:18 — "quem depende entra em pânico" |
| L2 — "mas isso acontece comigo?" | 0:18 — perguntas de identificação | 0:33 — "você funcionando pro app" |
| L3 — "então o que é controle de verdade?" | 0:33 — implícito após o reframe | 0:45 — definição de controle real |

**ELEMENTOS NOTÁVEIS (4 dos 8):**
- ✓ Disrupção / Polêmica (hook + reframe)
- ✓ Identificação (perguntas específicas — ansiosa no restaurante)
- ✓ Contra-intuitivo (o app deveria dar controle, mas está controlando você)
- ✓ Reframing ("é você funcionando pro app")

**ESTRUTURA EMOCIONAL:**

| Momento | Emoção |
|---------|--------|
| Hook | Disrupção — "espera, o quê?" |
| "a ferramenta pode ter seu lugar" | Alívio — "ela não tá me julgando" |
| "quem depende entra em pânico" | Reconhecimento |
| Perguntas de identificação | Identificação direta |
| "você funcionando pro app" | Consciência + leve desconforto |
| Controle de verdade | Clareza, possibilidade |
| CTA | Abertura, pertencimento |

**TEMPO ESTIMADO:**
~135 palavras @ ritmo conversacional de autoridade calma (~2,4 palavras/s com pausas) = **55-58 segundos**
