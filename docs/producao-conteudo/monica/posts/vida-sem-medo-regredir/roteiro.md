# Roteiro — vida-sem-medo-regredir

## Cabeçalho

- **Formato:** Carrossel (10 slides)
- **Tema:** Como é a vida de uma mulher que não tem mais medo de regredir
- **Pilar:** A Mulher 3R
- **Tese:** Não ter mais medo de regredir não significa viver numa bolha. É o sistema interno que mudou — não a vida ao redor.
- **Crença/visão construída:** A transformação é real e possível. Não é força de vontade. É método.
- **Total de slides:** 10
- **Versão atual:** v2 — 17/06/2026

### SLIDE 2 — FRAME

**Texto:**

Vai às mesmas festas.
Come nos mesmos restaurantes.
Tem os mesmos dias difíceis.

Fundo claro. Texto centralizado. Espaçamento generoso entre os blocos — cada grupo de frases respira. "É o sistema interno." em negrito ou tamanho levemente maior.

**Observação de legibilidade:** não juntar num único parágrafo. O ritmo de leitura é parte do impacto.

## Análise Técnica

**LOOPINGS:**

| Looping | Onde abre | Onde fecha |
|---------|-----------|-----------|
| L1 — "como é essa vida?" | S1 — título cria a pergunta | S9 — última cena completa o quadro |
| L2 — "mas como ela chegou aqui?" | S2 — "o sistema interno" (implícito) | S10 — "é método" |

**ELEMENTOS NOTÁVEIS (4 dos 8):**
- ✓ Identificação invertida (persona reconhece o que não tem — e deseja)
- ✓ História magnética (7 cenas reais e específicas)
- ✓ Valor prático (S2 mostra que é possível — sistema, não magia)
- ✓ Contra-intuitivo (S9 — ela ainda pensa em comida, mas mudou a emoção)

**ESTRUTURA EMOCIONAL:**

|-------|----------------|
| S1 | Curiosidade / Desejo |
| S2 | Orientação — "não é magia, é algo real" |
| S3 | Desejo + Reconhecimento invertido |
| S4 | Desejo — "imagine não abrir o app" |
| S5 | Alívio vicário |
| S6 | Leveza — cena mais simples, mais poderosa |
| S7 | Reconhecimento profundo |
| S8 | Libertação |

## Conexão com o Método 3R

Este carrossel é o destino da jornada. Cada cena mapeia uma das etapas:

- **RECONHECER:** S5 — ela sabe o que está acontecendo, não entrou em pânico com a balança
- **REORGANIZAR:** S4, S8 — a relação com a comida foi reorganizada (restaurante, viagem)
- **RECONSTRUIR:** S3, S6, S7, S9 — a confiança foi reconstruída (festa, convite, sábado, pensamento)

S10 fecha sem vender: "não é força de vontade, é método" — convida sem pitch.
