# Squad LPago Arcane

Squad de Lancamento Pago. **5 agentes** com personalidade propria:

- 🎯 **Atlas** — Estrategista (Chief) | doc mestre + consultoria + revisao de plano
- 📄 **Quill** — Copy & Pagina | briefing + copy redigida (12 secoes)
- 📺 **Spark** — Anuncios | roteiros + direcao criativa + edicao
- 💬 **Echo** — Copywriter de Mensagens | antecipacao + evento + recuperacao + downsell
- 📊 **Pulse** — Analista de Dados | diagnostico + retrospectiva

Conduz o usuario do plano (documento mestre + Calculadora Arcane) ate o pos-evento (recuperacao + retrospectiva), passando por pagina com copy redigida, anuncios, comunicacao multi-canal e diagnostico em tempo quase real.

## O que faz

- Constroi documento mestre (proposta + publico + orcamento via Calculadora Arcane + cronograma + metas)
- Redige briefing + copy final da pagina de vendas (12 secoes ou variante)
- Roteiriza 3-6 anuncios (Hook + Interesse + Oportunidade + CTA) com direcao criativa e de edicao
- Redige comunicacao completa (antecipacao pre-evento + dia do evento + recuperacao pos-evento)
- Diagnostica indicadores em tempo real (CPA, CTR, Hook Rate, conversoes) e prescreve acao priorizada
- Encerra ciclo com retrospectiva (faturamento, ROAS, aprendizados pro proximo lancamento)

## Para quem

Neutro — Euriler operando proprio lancamento OU aluno da Mentoria Arcane operando lancamento proprio.
