> Bússola do método. 3 métricas financeiras em formato de funil. Cada uma diagnostica uma camada da operação.

## As 3 Métricas (decoradas)

| Métrica | Threshold | Diagnostica |
|---------|-----------|-------------|
| **CVP** (Custo Visualização Página de Destino) | **4%** do produto | CRIATIVO |
| **Finalização** (Custo Initiate Checkout) | **23%** do produto | PÁGINA |
| **CPA** (Custo por Aquisição) | **45%** do produto | OFERTA |

**ROI alvo:** ≥ 2.0 (ROAS — coloca R$1, volta R$2).

## Connect Rate (métrica secundária)

Fórmula: Visualização Página de Destino / Cliques no Link.

| Faixa | Status | Ação |
|-------|--------|------|
| ≥ 85% | Ideal | Tudo OK |
| 75-84% | Aceitável | Monitorar |
| 50-74% | Problema | Investigar copy/relevância/velocidade |
| < 50% | **Crítico** | **HOSPEDAGEM** — trocar pra Rochost |

---

## ROI / ROAS

| ROI | Status | Ação |
|-----|--------|------|
| ≥ 3.0 | **Excelente** | Escalar agressivamente. +50% via Escala Máxima |
| 2.5 - 3.0 | **Alvo / Validado** | Criativo campeão. Pode escalar |
| 2.0 - 2.5 | **Aceitável** | Manter |
| 1.8 - 2.0 | **Mínimo** | Monitorar. Abaixo de 1.8 = desligar |
| 1.0 - 1.8 | **Zona de perigo** | Investigar |
| < 1.0 (período máximo) | **Desligar** | Não está pagando |

---

## Criativo Campeão

| Critério | Valor |
|----------|-------|
| Vendas mínimas | 5-10 (10 = estatisticamente confiável) |
| ROI | ≥ 2.5 |
| Funil 3X | Todas as 3 OK |
| Mínimo pra escalar | 4 criativos validados |
| Tempo ideal criativo | 40s a 1min10s |
| Volume mínimo/semana | 5-10 por oferta |

## Cenários de Diagnóstico

| Cenário | CVP | Final | CPA | Diagnóstico |
|---------|-----|-------|-----|-------------|
| **Erro Crônico** | Estourada | Estourada | Estourada | Criativo puxa tudo |
| **Página Fraca** | OK | Estourada | Estourada | Página não converte |
| **Checkout/Preço** | OK | OK | Estourada | Pessoa entra mas não compra |
| **Tudo OK + sem vendas** | OK | OK | OK | Volume baixo / pixel novo |
| **Desconexão Criativo-Página** | Barato | Cara | — | "Convido pra churrasco, chego e é pizza" |
| **Falso Funil 3X** | Barato | Barato | OK | Amigos clicando (Insta poucos seguidores) |

## Regras de Tempo (decisão)

| Regra | Detalhe |
|-------|---------|
| **Regra 48h / 3 dias** | Mín 2-3 dias completos antes de qualquer decisão |
| **Fase 3D** | Primeiros 3 dias = só métricas, não vendas |
| **Janela completa** | Período máximo + 7d + 3d antes de matar |
| **Exceção 24h** | Pixel aquecido + oferta validada + caixa |
| **Gramado** | Mín 5 dias pra avaliar |

---

## CPM (métrica externa que pode invalidar tudo)

- Anunciante NÃO controla
- Sazonalidade (Black Friday, Natal) infla
- Solução: criativos mais fortes pra compensar
- Caso Cleusa: trocou conta = CPM 90 → 18

---

## Sazonalidade

| Período | Comportamento |
|---------|---------------|
| **Jan-Mar** | MELHOR — "Ano novo, vida nova" |
| **Fev** | Um dos melhores meses |
| **Abr-Set** | Normal |
| **Out-Dez** | PIOR — Black Friday infla CPM |
| **Nov** | Pior mês do LT |

## Configuração da Coluna no Gerenciador

Apagar TODAS as colunas padrão e configurar pré-definição:

1. Visualização da página de destino (qty)
2. Custo por visualização (CVP)
3. Finalização de compra (qty)
4. Custo por finalização
5. Resultado / compras (qty)
6. Custo por resultado (CPA)
7. ROAS de compra
8. Connect Rate (calculada)
9. Data de criação

Salvar como pré-definição.

---

## Subir Preço — Quando

**Por métricas estouradas (com estudo):** CVP encaixa em produto maior. Testou 10+ criativos sem baixar.

**Por ROI aberto:** Métricas muito abaixo dos thresholds, há margem.

⚠️ **NÃO subir preço por preguiça** — só após esforço real.
