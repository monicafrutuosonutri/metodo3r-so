## Como Consultar

Quando agente precisa decidir algo importante:

1. **Aprovar oferta?** → `principios-nucleares.md` (P8 farmacia, P6 nao apaixone, P7 ruminacao)
2. **Aprovar headline?** → `anti-padroes.md` (AP1 Copy Dilma, AP9 Gourmesao)
3. **Diagnosticar metricas?** → `funil-3x-thresholds.md` + `anti-padroes.md`
4. **Falar com aluno?** → `voice-dna.md` + `frases-lei.md`
5. **Em duvida?** → consulta `principios-nucleares.md`
