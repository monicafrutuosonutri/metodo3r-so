# Templates — Páginas de Vendas Low Ticket

> Templates oficiais Maxxima de página de vendas low ticket. Modelos prontos pra estudar visual e estrutura.

## Arquivos

### `template-maxxima.html` (69KB)

Template HTML completo da página padrão Maxxima — **abre em qualquer browser**.

**Como usar:**

1. Dar duplo-clique no arquivo → abre no Chrome/Safari
2. Navegar a página inteira pra ver:
   - Estrutura visual (hero, antes/depois, conteúdo, oferta, FAQ, footer)
   - Hierarquia tipográfica (Poppins + Raleway)
   - Espaçamentos e larguras de bloco
   - Posicionamento de CTAs
   - Estilo de bullets, mockups, depoimentos

**Quem usa:**
- **Cole (`lt-page-master`):** referência visual ao montar página no Hostinger/WordPress/Elementor
- **Quinn (`lt-copywriter`):** referência de tamanho de cada bloco (quanto texto cabe onde) ao escrever copy

### `template-maxxima-oficial.json` (255KB)

Versão JSON do template (provavelmente Elementor / page builder exportado).

**Como usar:**
- Importar em Elementor (Templates → Import) pra reaproveitar layout pronto
- Ou inspecionar a estrutura de seções/widgets pra modelar
