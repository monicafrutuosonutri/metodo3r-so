## Diagnóstico por Funil 3X — A Bússola

O Funil 3X é a ferramenta central de diagnóstico do método. Cada etapa isola onde está o problema. Sem ele, qualquer análise é achismo.

### Estrutura do Funil 3X

| Etapa | Métrica | Meta | O que indica |
|-------|---------|------|-------------|
| **Etapa 1** | Custo por Visualização de Página de Destino | ~4% do valor do produto (R$67 = R$2,68) | Qualidade do CRIATIVO + velocidade da PÁGINA |
| **Etapa 2** | Custo por Finalização de Compra | ~23% do valor do produto | Qualidade da PÁGINA (copy, entregáveis, design) |
| **Etapa 3** | CPA (Custo por Aquisição) | ~45% do valor do produto | CHECKOUT + PREÇO + OFERTA |
| **ROI Final** | Valor do Produto / CPA | >= 2.0 (mínimo sustentável), >= 2.5 (campeão) | Saúde geral da operação |

> Para detalhes completos do Funil 3X, ver **Vol 02 — Tráfego e Estrutura de Campanha**.

### Árvore de Diagnóstico

```
Connect Rate < 75%?
  └── SIM → PARAR. Resolver Connect Rate ANTES de qualquer análise
       └── Ver P7 (Connect Rate Baixo)

Etapa 1 ESTOURADA (custo de visualização alto)?
  └── SIM → Problema: CRIATIVO
       └── Criar mais criativos, variar formato/copy
       └── NÃO mexer em página antes de resolver criativo
       └── Ver P1, P3, P5, P6

Etapa 1 OK, Etapa 2 ESTOURADA?
  └── SIM → Problema: PÁGINA
       └── Revisar headline, entregáveis, mecanismo único, imagens
       └── Teste AB de headline (mesmas palavras-chave)
       └── Ver P8, P9

Etapa 1 OK, Etapa 2 OK, Etapa 3 ESTOURADA?
  └── SIM → Problema: CHECKOUT / PREÇO / OFERTA
       └── Testar preço, revisar checkout, verificar plataforma
       └── Ver P17, P18, P21

Todas OK mas ROI < 2?
  └── Problema: PREÇO muito baixo ou ESTEIRA incompleta
       └── Testar preço maior, implementar orderbumps/upsell
       └── Ver P17, P18
```

### Padrões de Diagnóstico do Maxxima — Como Ele Lê Métricas

São 8 princípios que Maxxima repete sessão após sessão. Quem internalizar estes padrões resolve 80% dos problemas sozinho.

1. **Sequência rígida:** Criativo PRIMEIRO → depois página → depois escala. NUNCA mexer em tudo ao mesmo tempo. [Fonte: Mod 07, Batch 5a síntese]
2. **Uma variável por vez:** Não mudar criativo E página simultaneamente — "não sabe o que funcionou." [Fonte: Mod 07, sessões múltiplas]
3. **3 dias mínimo:** Não analisar campanha antes de 3 dias completos. "Chá de camomila." [Fonte: Mod 07 Sessões #35, #49]
4. **Período MÁXIMO:** Para decidir se desliga criativo, olhar período TOTAL (máximo), não 3 ou 7 dias. [Fonte: Mod 07 Sessão #9]
5. **Funil 3X como bússola:** Cada métrica isola onde está o problema. Sem funil, é achismo. [Fonte: Mod 07, sessões múltiplas]
6. **Regra do criativo campeão:** Só é campeão se: dentro do Funil 3X + 5-10 vendas + ROI >= 2.5 [Fonte: Mod 07 Sessões #42, #65]
7. **"Se tá dando bom, não mexe.":** Criativo/campanha com ROI bom = não mexer. Criar novos em paralelo. [Fonte: Mod 07 Sessão #11]
8. **Varredura antes de decisão drástica:** "Antes de tomar decisões drásticas, a gente precisa fazer a varredura" — escanear tudo antes. [Fonte: Mod 07 Sessão #24]

# CATEGORIA 1: Problemas de CRIATIVO

## P1. Criativo Caro no Funil 3X (Custo por Visualização Alto)

**Sintomas:** Custo por visualização de página de destino acima do parâmetro (meta: ~4% do valor do produto, ex: R$2,68 para produto de R$67). Funil 3X "travado na etapa 1".

**Diagnóstico:** Criativo não conecta com a ruminação do público. Hook fraco, copy genérica, formato errado, ou falta de dinamismo visual. Se o custo de visualização está alto, o problema é o criativo — ponto final.

**Solução:**
- Criar mais criativos seguindo PRSA (Problema, Rota, Solução, Ação)
- Variar formato mantendo a mesma copy: quadro, papel, caderno, iPad, vidro, grafite, flip chart
- Usar formatos que provaram funcionar: narrado, perguntas e respostas, filmando tela, caixinha de pergunta, tela dividida
- Se criativo entra caro, todo o funil fica caro — resolver criativo ANTES de mexer em página

> "Se o custo de visualização está alto = criativo ruim. Não olhar página antes de resolver criativo." [Fonte: Mod 07 Sessão #14]

> "Se entra caro, finaliza caro." [Fonte: Mod 07 Sessão #34]

**Casos reais:**
- **Luciano (Preço da Soja):** R$3,42 vs ideal R$2,68 — criativo estilo "Casas Bahia" refez para narrativo [Fonte: Mod 07 Sessão #48]
- **Cida (Checklist RH):** R$3+ de visualização — funil inteiro contaminado [Fonte: Mod 07 Sessão #34]
- **Nádia/Liliana (emagrecimento):** R$9-20 vs meta R$2,80 — combinado com Connect Rate baixo [Fonte: Mod 07 Sessões #14, #16]

## P2. Criativo com Click mas Sem Conversão (Criativo Clickbait)

**Sintomas:** Custo por clique barato, custo por visualização bom, mas nenhuma finalização de compra. ROI negativo apesar de métricas iniciais promissoras.

**Diagnóstico:** Criativo atrai curiosos, não compradores. Formato atrai perfil de "vitrinistas" (window shoppers) em vez de compradores reais. Ou criativo desconectado da promessa da página.

**Solução:**
- Criativo DIRETO pode estourar no Funil 3X mas converter melhor (qualidade de lead) — não descartar só porque está "caro"
- Diferentes criativos atraem diferentes perfis de comprador: um pode ter 22 finalizações e 2 compras; outro pode ter 18 finalizações e 8 compras
- Não tentar consertar o criativo de baixa conversão — focar em replicar o de alta conversão
- Criativo deve espelhar a promessa da página — conexão criativo-página é obrigatória

> "Criativo determina perfil do comprador. Diferentes formatos de criativo atraem pessoas fundamentalmente diferentes." [Fonte: Mod 07 Sessão #24]

**Casos reais:**
- **Thiele (Menopausa):** Um criativo teve 22 checkouts e 2 compras vs outro com 18 checkouts e 8 compras — formato leitura/diálogo atraiu pessoas mais qualificadas [Fonte: Mod 07 Sessão #24]
- **Renato (medicina/IA):** Criativo direto a R$5 estourava no Funil 3X mas convertia absurdamente (9 finalizações, 7 vendas) [Fonte: Mod 07 Sessão #40]
- **Ramon (Atividades Infantis):** 950K seguidores, custo de visualização muito baixo = clickbait [Fonte: Mod 07 Sessão #56]

## P3. Taxa de Assertividade de Criativo Muito Baixa

**Sintomas:** 15-20+ criativos testados, apenas 1-2 performam. Desperdício de verba alto.

**Diagnóstico:** Criativos sendo criados "do sovaco" (improvisados sem consultar a página/promessa). Falta de modelagem dos criativos que funcionaram. Desconexão entre copy do criativo e promessa da página.

**Solução:**
- Criativo DEVE espelhar a promessa da página — PRSA extraído da página de venda
- Modelar criativo vencedor: manter copy, variar formato/cenário/roupa
- Testar formatos variados do mesmo criativo: papel na mesa, papel caindo, objeto na mão, mensagem WhatsApp
- Choque de criativo: outra pessoa gravar (ex: esposa, parceiro) com voz e rosto diferentes
- Usar biblioteca de anúncios do Meta para modelar concorrentes

> "Vocês criam a promessa na página e depois gravam criativo do sovaco." [Fonte: Mod 07 Sessão #46]

**Casos reais:**
- **Robson:** 20+ criativos testados, apenas 1 performou — modelou o vencedor variando formato [Fonte: Mod 07 Sessão #29]
- **Pedro Sanches:** 13 criativos, oferta de finanças — choque de criativo com esposa gravando [Fonte: Mod 07 Sessão #50]
- **Tamiris:** Criativos de emagrecimento com hipnose não conectavam até mudar headline para ruminação direta [Fonte: Mod 07 Sessão #12]

## P4. Criativo Valida no Teste mas Não Escala

**Sintomas:** Criativo performa bem em ABO teste (ROI 2+), mas ao jogar em campanha de escala (CBO, Gramado) o ROI despenca.

**Diagnóstico:** Instabilidade da Meta, período sazonal (Black Friday inflando leilão), cluster ruim, BM com problema, sobreposição de audiência, ou criativos insuficientes para escala (precisam de 4 mínimo).

**Solução:**
- Criar conta de anúncio nova (BM nova, pixel novo) — CPM pode cair drasticamente (caso: de R$90 para R$18)
- Esperar período sazonal (Black Friday infla tudo: outubro-novembro é pior período)
- Testar cada estrutura de escala separadamente (cada oferta tem afinidade com estrutura diferente)
- Precisa de mínimo 4 criativos validados ANTES de qualquer tentativa de escala

> "Fase Vermelha — quando volta da otimização para teste. Indica criativos fracos para escala." [Fonte: Mod 07 Sessão #13]

**Casos reais:**
- **Cleusa (clareamento íntimo):** R$88k em 1 mês, criativos validam mas escala não funciona — CPM caiu de R$90 para R$18 ao trocar conta [Fonte: Mod 07 Sessão #58]
- **Matheus Castro:** Subiu 29 anúncios de uma vez, queimou ROI — "29 anúncios de uma vez = você é doido" [Fonte: Mod 07 Sessão #74]
- **Criativo com 16 vendas morreu ao escalar** — sobreposição de clusters [Fonte: Mod 07 Sessão #67]

## P5. Copy Dilma (Copy Confusa / Sem Clareza)

**Sintomas:** Headline confusa, sem ruminação direta. Criativo/página não comunica promessa clara. Múltiplas ruminações misturadas. Linguagem técnica em vez de linguagem do público.

**Diagnóstico:** Falta de EDI (Estudo de Diálogo Interno). Tentativa de resolver vários problemas ao mesmo tempo. Uso de jargão técnico ("déficit calórico", "superimunidade", "visagismo").

**Solução:**
- UMA ruminação por produto — produto que resolve vários problemas é um problema
- Headline com 3 palavras-chave da promessa, criar variações mantendo elas
- Usar a palavra que o público usa: "redução de caloria" > "déficit calórico", "imunidade alta" > "superimunidade"
- Pesquisar ruminações nos comentários de concorrentes (EDI)
- Copy "mastigadinha": 2+2=4 para o comprador

> "Copy Dilma = copy confusa, sem clareza, sem ruminação direta." [Fonte: Mod 07 Sessão #12]

**Casos reais:**
- **Aluno procrastinação:** Copy Dilma identificada na headline — reescrita para "Ainda essa semana comece a chegar ao fim do dia com tempo sobrando e sentimento de missão cumprida" [Fonte: Mod 07 Sessão #15]
- **Tamiris:** Copy dos criativos de emagrecimento classificada como "Copy Dilma" [Fonte: Mod 07 Sessão #12]
- **Marcelo (Visagismo):** Ruminações erradas na página — "diagnóstico confiável" em vez de "quer ir no encontro mas se veste feio" [Fonte: Mod 07 Sessão #37]

## P6. Criativo com Formato Errado / Falta de Rosto

**Sintomas:** Criativos com IA, animação, ou texto puro não convertem. Falta de autenticidade. Criativos tipo "Jornal Nacional" (expert falando estático pra câmera).

**Diagnóstico:** Low ticket exige autenticidade e dinamismo. Criativos sem rosto humano perdem credibilidade. Formato JN (estático na câmera) é ruim para low ticket.

**Solução:**
- Criativo precisa de ROSTO — autenticidade converte
- Formato narrativo > formato estático (regra geral, com exceções)
- "Não faz com IA não, tá tendo imperador sofrendo com isso"
- Formatos que funcionam: filmando tela, mostrando produto na mão, cenário contextual (frente de loja, dentro do carro, concessionária)
- Exceção: criativo estático simples feito no ChatGPT pode funcionar — vale testar

> "Criativo feio curto e direto vende melhor que produzido." [Fonte: Mod 07 Sessão #44]

**Casos reais:**
- **Guilherme (aluguel):** Criativo estático foi o que mais vendeu (contra-intuitivo — imagem superou vídeos) [Fonte: Mod 07 Sessão #44]
- **Giovanna (maquiagem):** Formato "personagem" (diálogo com colega) funcionou melhor que narrativo puro [Fonte: Mod 07 Sessão #42]
- **Leandro (mecânicos):** Expert não pode aparecer — gravar na frente de oficina, fundo verde, dentro do carro [Fonte: Mod 07 Sessão #53]

# CATEGORIA 2: Problemas de PÁGINA

## P7. Página com Connect Rate Baixo

**Sintomas:** Connect Rate abaixo de 75% (ideal 85%). Custo por visualização de página alto apesar de cliques baratos. Pessoas clicam no anúncio mas não chegam na página.

**Diagnóstico:** Página carrega lento (>3 segundos). Problema de hospedagem (WordPress/Hostinger). Plugin pesado. Imagens não otimizadas.

**Solução:**
- Resolver Connect Rate ANTES de analisar qualquer outra métrica — sem ele, tudo é inválido
- Testar velocidade no GT Metrics
- Trocar hospedagem para Rochost se Hostinger/WordPress estiver lento
- Primeira carga deve ser <3 segundos (6+ segundos = destruído)
- Tirar plugins desnecessários (ex: plugin de WhatsApp tira pessoa do fluxo)

**Fórmula:** Connect Rate = Visualização de Página de Destino / Cliques no Link

> "Fix Connect Rate first before analyzing anything else — no point optimizing page/creative if people aren't arriving." [Fonte: Mod 07 Sessão #19]

**Casos reais:**
- **Letícia:** Connect Rate 40-51% em TODOS os produtos — página com 6 segundos de first load, 11 segundos full load [Fonte: Mod 07 Sessão #19]
- **Nádia/Liliana:** Connect Rate 50-68% — problema técnico, encaminhou para Jeff [Fonte: Mod 07 Sessões #14, #16, #18]
- **Fernanda (inglês crianças):** Connect Rate 38% — recomendação: trocar hospedagem para Rochost [Fonte: Mod 07 Sessão #57]
- **Letícia (retorno):** Depois de trocar página, Connect Rate normalizou [Fonte: Mod 07 Sessão #26]

## P8. Página Não Converte (Criativo Bom, Página Ruim)

**Sintomas:** Criativo entra barato no Funil 3X, mas custo por finalização/CPA estoura. Visualização de página barata, compra cara.

**Diagnóstico:** Headline sem ruminação forte. Entregáveis fracos/genéricos. Falta de mecanismo único visível. Imagens ruins. Estrutura fora do método. Falta de prova social.

**Solução:**
- Diagnosticar via Funil 3X: se entrada barata e saída cara = problema de página (não criativo)
- Checar headline: deve ter ruminação + mecanismo único + fator tempo
- Checar entregáveis: mínimo 4 + 3 bônus (sem chamar de "bônus")
- Mockup + Título + Descrição para cada entregável (não usar ícones)
- Imagens VERTICAIS (mobile-first) — horizontal fica pequena no celular
- Usar template do Maxxima 100% — se fugiu, é mais uma variável a diagnosticar
- Teste AB: duplicar página, mudar 1 variável, rodar simultaneamente

> "Quando olho o funil com criativo de 1.29 e página em 30, o que tem nessa página? Funil 3X é uma bioimpedância — sem ele, análise é abstrata." [Fonte: Mod 07 Sessão #66]

**Casos reais:**
- **Suzy (emagrecimento):** Criativo R$1,29 mas página R$30 de custo — headline reformulada ao vivo. Problemas: "bússola do Noé", "balança da feira", "mulher gorda sem cintura" nas imagens [Fonte: Mod 07 Sessão #66]
- **Giovanna (maquiagem):** 62 criativos testados em 2 meses, criativos entrando barato mas página estourando [Fonte: Mod 07 Sessão #42]
- **Alisson:** Criativos bons mas página estourou — pensou em testar página antiga vs nova [Fonte: Mod 07 Sessão #47]

## P9. Oferta Aberta / Falta de Mecanismo Único

**Sintomas:** Página com promessa genérica. Headline sem especificidade. Produto tenta resolver tudo (hospital, não farmácia). Sem tangibilidade.

**Diagnóstico:** Oferta larga demais, sem nichar. Mecanismo único ausente ou invisível. Promessa sem "como" concreto. "Aprenda" em vez de "copie e cole".

**Solução:**
- Mecanismo único OBRIGATÓRIO: planilha, ciclo de orações, protocolo, kit, checklist, guia — algo tangível
- "3 passos" NÃO é mecanismo — orações, ciclo, roteiro, calculadora = mecanismo
- Nichar a promessa (Pirâmide de Maslow: dinheiro > saúde > relacionamento)
- Low ticket = band-aid/fast food, não transformação — alívio imediato
- "Se está explicando muito o low ticket, não é mais low ticket"

> "Oferta aberta = ela não vinga. Sem mecanismo único, é só conselho genérico." [Fonte: Mod 07 Sessão #74]

**Casos reais:**
- **Rotina da mulher organizada:** Faltava mecanismo tangível — "Use essa lista de tarefa" estava missing [Fonte: Mod 07 Sessão #74]
- **Pedro (cachorro/adestramento):** Headline "descubra como acabar com xixi fora do lugar" — mas COMO? Falta mecanismo [Fonte: Mod 07 Sessão #45]
- **Marcos (Intimidade com Deus):** "3 passos simples" não sustenta — solução: "ciclo de orações" como mecanismo [Fonte: Mod 07 Sessão #50]
- **Natália (atração de clientes):** "20 maneiras simples para dobrar clientes" — "Parece título de blog do Sebrae" [Fonte: Mod 07 Primeiro Encontro T4]

# CATEGORIA 3: Problemas de TRÁFEGO / CAMPANHA

## P10. Estrutura de Campanha Errada

**Sintomas:** Múltiplos criativos num conjunto. 118 conjuntos na mesma campanha. Gestor de tráfego de lançamento operando perpétuo. Orçamento distribuído errado.

**Diagnóstico:** Confusão entre teste de criativo e escala. Gestores de lançamento não entendem perpétuo. Falta de isolamento (1 campanha > 1 conjunto > 1 anúncio).

**Solução:**
- Estrutura correta para TESTE: 1 campanha > 1 conjunto > 1 anúncio (isolado)
- Orçamento teste: 45% do valor do produto por dia por conjunto (R$67 = R$30/dia)
- Campanha de teste =/= campanha de escala — estruturas completamente diferentes
- Nunca repetir mesmo criativo em clusters diferentes — se deu errado em 1, dá errado em todos
- Subir campanhas à meia-noite e três (00:03)

> "Quem é gestor de lançamento não é necessariamente gestor de perpétuo — skills diferentes." [Fonte: Mod 07 Sessão #35]

**Casos reais:**
- **Felipe/Arthur:** 118 conjuntos em 1 campanha desde dezembro, mesmo criativo repetido em clusters — equipe profissional de tráfego fez errado. R$1.389 investido, R$938 retorno [Fonte: Mod 07 Sessão #35]
- **Aluno com gestor:** Usando R$10 por conjunto com vários criativos dentro — "seu gestor fez errado" [Fonte: Mod 07 Sessão #44]
- **Matheus Castro:** 29 anúncios de uma vez — "você é doido" [Fonte: Mod 07 Sessão #74]

## P11. Desligar Campanha/Conjunto Prematuramente

**Sintomas:** Conjuntos com ROI 2-3 desligados antes de maturar. Advantage duplicada com R$45 gastos e desligada. Desespero com oscilação de ROI.

**Diagnóstico:** Falta de paciência. Confusão entre oscilação natural e queda real. Gestor de tráfego incompetente. Não entender a "Dança do ROI".

**Solução:**
- Esperar 3 dias mínimo antes de analisar campanha ("chá de camomila")
- CBO precisa de pelo menos 3 dias para engrenar
- Gramado precisa de 5 dias mínimo para avaliar
- Regra de desligar: olhar período MÁXIMO (todo período), não 3 ou 7 dias
- Nunca desligar conjunto com ROI > 2 sem motivo forte
- Se criativo absurdamente caro já no 1o dia (R$15-20): exceção — pode pausar

> "Falta de tomada de decisão está custando muito dinheiro." [Fonte: Mod 07 Sessão #11, sobre Keuri/Débora com ROI 6 no chinelo não escalando]

**Casos reais:**
- **Keuri/Débora:** Advantage duplicada com R$45 gastos e desligada — mataram muito cedo [Fonte: Mod 07 Sessão #11]
- **Thiago:** Gestor de tráfego desligando conjuntos com ROI 2-3 prematuramente — erro grave. Reativou e confirmou que criativos eram bons [Fonte: Mod 07 Sessões #11, #17]
- **Grace (vinhos):** Criativo de R$6,23 rodando há menos de 2 dias — "chá de camomila, esperar 3 dias" [Fonte: Mod 07 Sessão #35]

## P12. CPM Altíssima

**Sintomas:** CPM muito acima de R$30. Custo por mil impressões desproporcional. Criativos novos já nascem caros.

**Diagnóstico:** Problema de conta/pixel. Conta de anúncio "suja" ou com histórico ruim. Período sazonal (Black Friday). BM problemática.

**Solução:**
- Conta nova + pixel novo (solução mais frequente)
- Testar mesmos criativos em conta de anúncio diferente — CPM pode variar drasticamente entre contas
- BM não é tempo que faz boa — é teste. BM nova pode funcionar melhor que velha
- Esperar passar período sazonal (Black Friday infla tudo)
- Campanha de Reconhecimento/Alcance primeiro para aquecer conta nova

**Casos reais:**
- **Caio (Queima Gordura):** CPM altíssima — solução: pixel novo + conta nova [Fonte: Mod 07 Sessão #50]

## P13. Escala Destruindo ROI (Síndrome do Reset e Falta de Ritmo)

**Sintomas:** ROI positivo em teste, negativo ao escalar. Tentar escalar com 1-2 criativos. Subir toda verba de uma vez. Resetar o que funciona para implementar método novo.

**Diagnóstico:** Escalar sem criativos suficientes (mínimo 4). Subir verba agressivamente demais. Não entender o ritmo luta/glória. Síndrome do Reset — destruir o que funciona.

**Solução:**
- Mínimo 4 criativos validados ANTES de qualquer escala
- Regra dos 20%: escalar orçamento máximo 20% por dia (subir verba às 23h-23h30, nunca cedo)
- Ritmo "3 Dias de Luta, 4 Dias de Glória": 3 dias testando/arriscando, 4 dias só vencedores
- Nunca duplicar orçamento — aumento gradual
- Fazer mudanças em teste AB, nunca resetar tudo
- Escalar pelo lucro: "Quanto estou tendo de lucro? R$500/dia? Subo R$500 e testo"
- Testar CADA estrutura de escala separadamente (ABO Raiz, CBO 500, Gramado, Mix de Criativo)

# CATEGORIA 5: Problemas TÉCNICOS

## P19. Pixel Não Marcando Eventos / Pixel Invertido

**Sintomas:** Vendas na Hotmart mas não aparecem no gerenciador. Eventos invertidos (mais finalização que visualização). Discrepância grande entre plataforma e Meta.

**Diagnóstico:** Pixel instalado errado. API de conversão não configurada. Pixel só na página mas não no checkout (ou vice-versa). Eventos errados configurados na Hotmart. Hotmart reportando PIX gerado (não pago). PixelYourSight com problemas recorrentes.

**Solução:**
- Pixel precisa estar TANTO na plataforma de checkout (Hotmart) QUANTO na página de vendas
- Configurar API de conversão + eventos: View Content, Initiate Checkout, Purchase
- API token =/= Pixel ID — token vai para Hotmart, Pixel ID vai para página
- Script no HEAD da página (não body)
- Verificar via Meta Pixel Helper no navegador
- Dizer para Hotmart NÃO reportar PIX gerado não pago
- Após corrigir pixel: DESLIGAR campanhas e criar NOVAS (métricas não retroagem)
- Considerar PixelCat ou pixel manual no HTML em vez de PixelYourSight

**Checklist de configuração de pixel:**

| Componente | Onde | O que configurar |
|-----------|------|-----------------|
| Pixel ID | Página de vendas (HEAD) | Script do pixel no código da página |
| API Token | Hotmart/Plataforma | Token de conversão (NÃO é o Pixel ID) |
| Eventos | Hotmart | View Content + Initiate Checkout + Purchase |
| PIX gerado | Hotmart | Desmarcar "reportar PIX gerado não pago" |
| Verificação | Navegador | Meta Pixel Helper (extensão Chrome) |

**Casos reais:**
- **Rodrigo/Rogério:** Pixel configurado errado na Ticto — faltava pixel no checkout (só na página). API não estava ativa [Fonte: Mod 07 Sessão #49]
- **Pedro/Lara (corretagem):** Pixel não marcando eventos de página/checkout [Fonte: Mod 07 Sessão #47]
