# Tom por Tipo de Tema — Swipe File

> Tabela de tom recomendado por categoria de tema. Consultar **antes** do protocolo de protótipos (`*prototipos`) pra calibrar o vetor inicial do batch.

---

## Quadro Principal

| Tema | Tons dominantes | Estilos de abertura recomendados | Sintomas que indicam tom errado |
|------|------------------|----------------------------------|----------------------------------|
| **Sensível** (vergonha, culpa, vazio íntimo, religiosidade, identidade) | Acolhedor + vulnerável + cinematográfico | A (confissão) + B (observação) + C (cena) + D (pergunta gentil) + F (tabu) | Camelô, infomercial, acusação direta, listão em rajada |
| **Agressivo** (frustração, raiva, injustiça, "tá farta") | Confrontador + diagnóstico | D (pergunta confrontadora) + estilos diretos | Suave demais, condescendente, "amiguinho" |
| **Técnico** (resultado, performance, escala, ROI) | Convidativo + diagnóstico | D + E (com caso real numérico) | Emocional, vago, sem dado |
| **Universal cotidiano** (rotina, tempo, organização) | Acolhedor + cinematográfico | B (observação coletiva) + C (cena) | Apocalíptico, dramático demais |
| **Aspiracional** (estética, status, lifestyle) | Convidativo + cinematográfico | C (cena) + D (pergunta) | Vendedor, "guru", forçado |
| **Tabu nomeado** (sexo, dinheiro, vícios) | Vulnerável + reconhecedor | A (confissão) + F (vulnerabilidade no tabu) | Moralista, julgador, evasivo |

## Catálogo de Tons (7 registros)

| Tom | Quando usar | Exemplo de abertura |
|-----|-------------|---------------------|
| **Acolhedor** | Tema sensível, "você não tá sozinha" | "Tem uma coisa que toda [persona] já me disse..." |
| **Confrontador** | Tema agressivo, persona já farta | "Para de se enganar. [pausa] Você sabe que [X]..." |
| **Cinematográfico** | Cena cotidiana específica | "É uma quinta à noite. A casa silenciou..." |
| **Vulnerável** | Confissão própria desarma defesa | "Eu vou falar uma coisa que eu nunca tinha dito..." |
| **Conspiratório** | Quer revelar algo "que ninguém fala" | "Vou te contar uma coisa que ninguém fala..." |
| **Diagnóstico** | Tema técnico ou agressivo | "Isso tem nome. É [X]. E funciona assim..." |
| **Convidativo** | Convidar reflexão sem acusar | "Posso te perguntar uma coisa?..." |

## Combos de Tom (1-2 dominantes por batch)

### Tema sensível
```
Acolhedor + Vulnerável + Cinematográfico
```
- Abertura: A/B/C
- Rota: pode subir pra D ou F na metade
- Solução: volta pro acolhedor

### Tema agressivo
```
Confrontador + Diagnóstico
```
- Abertura: D confrontadora ("Quanto tempo você vai aguentar...")
- Rota: diagnóstico ("Isso acontece porque...")
- Solução: convidativo + tangível

### Tema técnico
```
Convidativo + Diagnóstico
```
- Abertura: D + número/dado
- Rota: mecanismo explicado
- Solução: prova social ou caso real

### Tema aspiracional (cuidado)
```
Convidativo + Cinematográfico
```
- NUNCA usar "guru/coach"
- Cena concreta > promessa abstrata

## Anti-padrões de Tom (sintomas de calibragem errada)

| Sintoma | Tema | Provável erro |
|---------|------|---------------|
| Persona se sente "rotulada" | Sensível | AP12 — etiqueta-rótulo repetida |
| Persona se sente sufocada | Sensível | AP13 — listão em rajada |
| Persona ativa defensividade no hook | Sensível | AP11 — pergunta-acusação |
| Persona acha "amiguinho" | Agressivo | Tom acolhedor em tema que pede confrontador |
| Persona "não vê o número" | Técnico | Tom emocional sem dado |
| Persona acha "vendedor" | Aspiracional | Tom confrontador em tema que pede convidativo |

## Mistura emocional dentro do batch (regra avançada)

Tom suave em **todos** os 10 PRSAs do batch gera **batch morno**. Mesmo em tema sensível, variar emoção entre os PRSAs:

| PRSA | Tom dominante sugerido |
|------|------------------------|
| #01-02 | Suave/acolhedor (estilo A ou B) |
| #03 | Cinematográfico (estilo C) |
| #04 | Pergunta gentil (estilo D) |
| #05 | Tabu nomeado (estilo F) |
| #06 | Confissão direta (estilo A) |
| #07 | Cena descritiva (estilo C) |
| #08 | Observação coletiva (estilo B) |
| #09 | Caso real (estilo E — só se tem caso) |
| #10 | Mix híbrido (estilos A+F) |

**Variedade emocional é o que diferencia 10 PRSAs entre si** — não 10 versões da mesma coisa.

## Pergunta de calibragem (antes do batch)

Antes de escrever, perguntar ao aluno:

1. Qual o **tema central** dessa oferta?
2. Que tom da persona da gravação **descarta de cara**? (ex: "ela não consegue ser confrontadora")
3. Tem **vídeo de referência** dela falando direto pra câmera? (mesmo stories antigo)
4. **Linguagem religiosa**: tonelada / sutil / nenhuma?
5. **Show do produto** cabe? (planner físico, tela, livro)
6. Permite **caso real** com nome citado?

Respostas → calibrar combo de tom + estilos de abertura → 2-3 protótipos → batch.
