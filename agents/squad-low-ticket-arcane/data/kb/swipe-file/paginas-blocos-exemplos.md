# Swipe File — Pagina 14 Blocos com Exemplos

> Os 14 blocos com funcao + template + exemplos reais + anti-padroes.
> Fonte: Vol 3 (Etapas 5, 7, 8 da Imersao + Mod 02).

---

## Visao Geral

Sequencia: **Abre Dor → Abre Desejo → Fechamento**

```
Dor (1-4) → Desejo (5-8) → Fechamento (9-14)
```

---

## Bloco 1 — Vender Sozinho (Primeira Dobra)

**Funcao:** A grande ideia central. Vende sozinho.

**Template:**
```
[Sobrancelha — faixa fina contextual]
[HEADLINE 3 PARTES costurada]

[Bullet 1: Mesmo que [objecao]]
[Bullet 2: Sem precisar [objecao]]
[Bullet 3: Independente de [objecao]]

[IMAGEM neurodesign — vertical, mobile-first]
```

**Regras:**
- SEM botao
- SEM foto expert primeiro plano (frio)
- SEM video de venda padrao
- DEVE conectar com criativo (anti-Sovaco)

**Exemplo (Mil Seguidores):**
```
[Sobrancelha: Comprovado 11,7% de conversao]

DESCUBRA COMO FAZER MIL SEGUIDORES OU MAIS POR SEMANA NO SEU INSTAGRAM

✓ Mesmo que voce tenha conta nova
✓ Sem precisar postar todo dia
✓ Independente do nicho

[imagem: print de Instagram com gráfico subindo]
```

## Bloco 2 — Depoimentos

**Funcao:** Provas sociais geram desejo ("eu quero isso tambem").

**Regras:**
- SEM CARROSSEL
- Layout 3+3 ou 2+2
- WhatsApp parece WhatsApp (balao + horario)
- Se nao tem ainda, OCULTAR e ativar quando tiver

**Template:**
```
[Headline costurada: "Veja pessoas que conseguiram [promessa]"]

[Depoimento 1 — print WhatsApp ou antes/depois]
[Depoimento 2]
[Depoimento 3]
[Depoimento 4]
[Depoimento 5]
[Depoimento 6]
```

## Bloco 3 — Dor Latente

**Funcao:** Identificacao com dores que JA existem na persona.

**Regras:**
- 4-6 ruminacoes Score 4-5 (max)
- Copy rapida, fast food
- Imagem REPRESENTANDO a dor

**Template:**
```
[Headline costurada: "Voce reconhece alguma dessas situacoes?"]

✓ [Ruminacao Score 5 — frase literal]
✓ [Ruminacao Score 5]
✓ [Ruminacao Score 5]
✓ [Ruminacao Score 4]
✓ [Ruminacao Score 4]

[IMAGEM: representa a dor]
```

**Exemplo (loja virtual):**
```
✓ Meu direct ta morto
✓ Posto e ninguem comenta
✓ Stories nao tem visualizacao
✓ Nao consigo achar quem comprar
```

## Bloco 5 — Passo a Passo

**Funcao:** Mostrar que e fast food.

**Regras:**
- 3-4 passos (max)
- NUNCA "modulo 1, modulo 2"
- Mockups ao lado

**Template (Diagnostico-Ponte-Resultado):**
```
[Headline costurada: "Esses sao os [N] passos rapidos pra [promessa]"]

PASSO 1: [Verbo + diagnostico]
[Mockup do entregavel desse passo]

PASSO 2: [Verbo + mecanismo/ponte]
[Mockup]

PASSO 3: [Verbo + resultado/promessa]
[Mockup]
```

**Exemplo:**
```
PASSO 1: Identifique o que ta travando suas vendas no Insta
PASSO 2: Aplique o protocolo de bio + stories
PASSO 3: Tenha mil seguidores em 7 dias
```

## Bloco 6 — Tudo que Vai Receber + Bonus (4+3)

**Funcao:** Mostrar valor + gerar desejo.

**Regras:**
- 4 entregaveis principais + 3 extras
- NUNCA "bonus" (linguagem proibida)
- Mockups REAIS (535x535px)
- "Imagem ilustrativa NAO vende"

**Template:**
```
[Headline costurada: "Veja tudo que voce vai receber PRA [promessa]"]

[ENTREGAVEL 1]
[Mockup 535x535]
[Titulo Food Porn]
Resolve: [beneficio em linguagem da persona]

[ENTREGAVEL 2-4 idem]

— Ainda nao acabou, voce tambem vai receber: —

[EXTRA 1]
[Mockup]
[Titulo Food Porn]

[EXTRA 2-3 idem]
```

## Bloco 7 — Pra Quem Serve

**Funcao:** Confirmacao final via dialogo interno.

**Regras:**
- Por DIALOGO INTERNO, NAO profissao
- Excecao: produto serve varias profissoes E e relevante (ex: Pack Bio)

**Template:**
```
[Nome do produto] e pra voce que:

✓ [ruminacao 1]
✓ [ruminacao 2]
✓ [ruminacao 3]
✓ [ruminacao 4]
✓ [ruminacao 5]

Se voce se identifica com ao menos [N] itens acima,
[nome do produto] e pra voce.
```

## Bloco 8 — Ancoragem de Preco

**Funcao:** Empilhamento de valor (~90% acima do preco real).

**Regras:**
- Precos individuais REAIS
- "Nao e pra ficar mentindo"
- Soma ancorada
- Max ~R$800-900 ancorado (nao absurdo)

**Template:**
```
Recapitulando tudo que voce vai receber:

✓ [Entregavel 1] ............ R$ [valor real]
✓ [Entregavel 2] ............ R$ [valor real]
✓ [Entregavel 3] ............ R$ [valor real]
✓ [Entregavel 4] ............ R$ [valor real]
✓ [Extra 1] ................. R$ [valor real]
✓ [Extra 2] ................. R$ [valor real]
✓ [Extra 3] ................. R$ [valor real]

No total, tudo isso deveria custar R$ [SOMA ~90% acima],
mas hoje voce vai ter acesso a tudo isso por apenas...

→ [Bloco 9]
```

## Bloco 9 — Primeiro Botao + Tudao

**Funcao:** PRIMEIRO botao da pagina inteira.

**Regras:**
- Botao **VERDE**, sem animacao
- Tudao 650x520 com TODOS os entregaveis empilhados
- Preco parcelado + a vista + CTA

**Template:**
```
[TUDAO 650x520 — todos entregaveis empilhados]

DE R$[ancorado]
POR APENAS

R$ [parcelado] em [N] vezes
ou R$ [a vista] a vista no PIX (com [%] de desconto)

[BOTAO VERDE: ACESSAR AGORA]
```

## Bloco 10 — Conversa Seria

**Funcao:** Cartada final pos-preco.

**Regras:**
- UNICO bloco com texto corrido (foge da copy dinamica)
- Conta matematica do prejuizo
- Confronta com 2 opcoes

**Template:**
```
Vamos ter uma conversa seria, [tipo persona].

[Conta matematica do prejuizo real, ex:]
"Quantas vezes voce abriu o WhatsApp esperando uma mensagem que
nao chegou? Quantas oportunidades de negocio voce perdeu nas
ultimas 4 semanas? R$150 por sessao x 24 semanas = R$3.600 perdidos."

Voce tem 2 opcoes agora:

OPCAO 1: Continuar como esta. Mais 30 dias com [dor], com Insta morto,
com [pior cenario].

OPCAO 2: Investir R$67 hoje e mudar isso definitivamente.

[Botao verde — repete CTA]
```

## Bloco 11 — Autoridade

**Funcao:** "Manhas" (provas praticas), nao "escambaux" (titulos).

**Regras:**
- Curto, sem "sobre mim"
- Foco em RESULTADOS concretos com NUMEROS
- "Tem quem tem MBA, tem quem tem doutorado escambau. E tem nos que tem as manhas"

**Template:**
```
Quem ta por tras do [produto]?

[Foto do expert SEM ser primeiro plano — segundo plano, neurodesign]

[NOME DO EXPERT] e responsavel por [RESULTADO CONCRETO COM NUMEROS,
ex: "ja ajudou +3.000 alunos a alcancar mil seguidores no Instagram"].

E agora ele/ela quer fazer com que voce chegue nesse resultado.
```

## Bloco 13 — FAQ

**Funcao:** **Matar medo de pagar**, nao duvidas genericas.

**Perguntas-tipo:**
```
Pergunta: O pagamento e seguro?
Resposta: Sim, [plataforma] e ...

Pergunta: Como vou ter acesso?
Resposta: Apos a compra, voce recebe email com login...

Pergunta: Tem garantia?
Resposta: A Hotmart te oferece 7 dias de garantia automatica...

Pergunta: Funciona no celular?
Resposta: Sim, todo conteudo e mobile-first...

Pergunta: Funciona pra meu nicho?
Resposta: Sim, ja tivemos alunos de...
```

Formato: accordion (expansivel).

## Bloco 14 — Rodape

**Funcao:** "Mostrar que e grandao."

**Regras:**
- Logo profissional
- Email **DOMINIO PROPRIO** (NUNCA Gmail)
- CNPJ
- "Todos os direitos reservados © [ano]"

**Template:**
```
[LOGO PROFISSIONAL]

Email: contato@dominio.com.br | CNPJ: XX.XXX.XXX/0001-XX

© [Ano] [Nome da Empresa]. Todos os direitos reservados.
```

- Primeira dobra com cor diferente

- Design clean, fundo bege claro
- Order Bump complementar
