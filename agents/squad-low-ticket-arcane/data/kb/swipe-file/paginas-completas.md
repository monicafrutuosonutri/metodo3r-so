- **PARTE 1 — 2 LPs completas** preservadas bloco a bloco (Hero → Antes/Depois → Empatia → Tentou de Tudo → Virada → Conteúdo → Galeria → Transformação → Qualificação → Bio → Recapitulando → Oferta → Carta Direta → Pós-compra → FAQ → Footer)

- **PARTE 2** — análise comparativa, padrões observados, estruturas de copy mais fortes (literais)
- **PARTE 3** — ESTRUTURA MASTER (ordem dos blocos numa LP de low ticket)
- **PARTE 4** — Copy-lego: headlines-promessa, qualificação, empatia/monólogo, anti-objeção, virada, bullets, ancoragem, preço, garantia, CTA, carta direta, comparativo binário, disclaimer footer
- **PARTE 5** — Diferenças low ticket vs lançamento pago

## Fidelidade

Ambas LPs extraídas via Playwright (DOM completo) + Apple Vision OCR (imagens/mockups). Texto direto do DOM = alta fidelidade. Textos em imagens = OCR limpo.

## Como usar (Quinn)

1. Antes de escrever copy nova: ler PARTE 3 (ESTRUTURA MASTER) pra fixar ordem dos blocos
2. Por bloco: ir na PARTE 4 (copy-lego) e pegar 3-5 referências do bloco específico
3. Pra fluxo completo: copiar o esqueleto de LT-1 ou LT-2 conforme nicho mais próximo
4. Adaptar vocabulário + mecanismo + ruminação ao produto do aluno
