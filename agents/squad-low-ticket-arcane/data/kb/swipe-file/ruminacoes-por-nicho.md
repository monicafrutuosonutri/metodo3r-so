# Swipe File — Ruminacoes Score 4-5 Por Nicho

> Frases REAIS validadas em sessoes ao vivo. Score 4-5 = "literalmente o que a persona pensa quando esta sozinha".
> Fonte: Vol 7 + Mod 03/03 + Mod 07.

---

## Como Usar

1. Identificar nicho do aluno
2. Pegar 4-6 ruminacoes Score 4-5 dessa lista (max na pagina)
3. Validar com EDI proprio (3 perguntas)
4. Adaptar/refinar com linguagem ESPECIFICA da persona

**REGRA:** Score 4-5 = frase LITERAL, intima, vulneravel. Se a pessoa nao "levanta a mao" = Score baixo, descarta.

## Como Construir Suas Proprias

### Comando ChatGPT (com filtro de linguagem popular)

```
Mulheres que sofrem com {DOR}, quais sao os pensamentos que ela esta
repetindo no dialogo interno dela com o comportamento cognitivo de
ruminacao mental?

[ADICIONAR SEMPRE:]
Escreva exatamente as frases do dialogo interno dela, frases que
brasileiros falam sem formalidade, com girias e palavroes e
expressoes reais.
```

### Onde Validar

| Fonte | Por que |
|-------|---------|
| **YouTube (comentarios)** | Pessoas expressam dor real sem filtro |
| **TikTok (videos virais)** | Linguagem crua, reacoes espontaneas |
| **Publico real + GPT** | Validacao direta |

### Score 0-5

| Score | Acao |
|-------|------|
| 0-2 | DESCARTA (generica) |
| 3 | REFINA |
| 4-5 | USA (literal, intima, vulneravel) |

**Max 4-6 iluminacoes na pagina.**

## Anti-Padroes — NÃO usar

| Anti | Por que |
|------|---------|
| "Preciso emagrecer" | Generico, nao e dialogo interno |
| "Quero ganhar dinheiro" | Sem especificidade |
| "Nao consigo resultados" | Vago demais |
| "Eu faco de tudo e nada resolve" | Linguagem chique (ChatGPT) |
| "Quero ser fitness" | Aspiracional, nao DOR |
| "Acalmar a mente" | Desejo aberto (vs "Zere ansiedade") |
