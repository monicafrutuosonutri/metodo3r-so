# Checklist: Analysis Completeness (QG-MF-002)

## Sintetica

- [ ] >= 20 KFs extraidos no total
- [ ] >= 2 KFs tipo FRAMEWORK (com nome + passos)
- [ ] >= 3 KFs tipo HEURISTIC (com SE/ENTAO)
- [ ] >= 3 dominios identificados
- [ ] >= 2 fontes/experts representados
- [ ] Confianca media >= 0.7
- [ ] Ratio inferred < 0.4
- [ ] Voice profile completo (tom, frases, vocabulario)
- [ ] Voz fundida proposta com expert dominante
- [ ] Convergencias identificadas (pelo menos 1)
- [ ] Tensoes documentadas (se houver)
- [ ] Gaps listados

## Consultor

- [ ] >= 15 KFs extraidos no total
- [ ] >= 2 KFs tipo FRAMEWORK (com nome + passos)
- [ ] >= 3 KFs tipo HEURISTIC (com SE/ENTAO)
- [ ] >= 3 subtopicos identificados
- [ ] Confianca media >= 0.7
- [ ] Ratio inferred < 0.4
- [ ] Voice profile completo
- [ ] Gaps listados

## Veto Conditions (bloqueiam avanço)

- Total KFs < 10
- Zero KFs tipo FRAMEWORK
- Ratio inferred > 0.4
- Voice profile ausente
