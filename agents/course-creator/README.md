# Course Creator

Squad de criacao de curso — da ideia ao produto empacotado + checklist de producao.

## Ativacao

```
/courseCreator
```

## Agentes

| Agente | Tier | Role |
|--------|------|------|
| @course-chief | Orchestrator | Pipeline, progress, quality gates, state management, review, checklist |
| @course-creator | Tier 1 | Benchmarking, metodologia, co-desenvolvimento de teoria, estruturacao, roteirizacao |
| @brand-architect | Tier 1 | Naming, posicionamento, identidade verbal |
| @product-packager | Tier 1 | Proposta de valor, pricing, cartao de identidade do produto |

## Pipeline (10 fases)

```
Fase 0: Setup (chief)                     ─── QG-CC-001: Curso definido
Fase 1: Benchmarking + PRD (creator)      ─── QG-CC-002: PRD aprovado
Fase 2: Ingestao Metodologia (creator)    ─── QG-CC-003: Metodologia mapeada
Fase 3: Estrutura Macro (creator)         ─── QG-CC-004: Macro validada
Fase 4: Estrutura Modulos (creator, loop) ─── QG-CC-005: Modulo validado
Fase 5: Aulas (creator, loop)             ─── QG-CC-006: Aula fechou
Fase 6: Branding (brand-architect)        ─── QG-CC-007: Branding aprovado
Fase 7: Empacotamento (product-packager)  ─── QG-CC-008: Produto empacotado
Fase 8: Revisao Final (chief)             ─── QG-CC-009: Review aprovado
Fase 9: Checklist Producao (chief)        ─── QG-CC-010: Checklist confirmado
```

## Ciclo Fractal

Em TODO nivel (curso, modulo, aula):

**CO-DESENVOLVER TEORIA -> ESTRUTURAR -> REFINAR -> VALIDAR**

## Escopo

**Inclui:** Da ideia ao produto empacotado + checklist de tudo que falta produzir pra lancar
**Nao inclui:** Gravacao, edicao, publicacao, pagina de vendas (esses saem no checklist)
