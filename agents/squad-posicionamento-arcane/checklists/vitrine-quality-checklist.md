# Checklist de Qualidade — Vitrine

> Usado pelo @vitrine-strategist em cada item antes de marcar como check.
> Derivado de `data/vitrine-instagram-2026.md` + `data/copy-de-oferta.md`.

---

## Item 1 — Display Name

- [ ] Padrao "Nome Sobrenome | Credencial de Especialista" respeitado
- [ ] ≤ 30 caracteres total
- [ ] Sem numeros (exceto se faz parte do nome real)
- [ ] Sem underscores excessivos
- [ ] Sem emojis decorativos (separador estilizado opcional)
- [ ] Credencial diz O QUE faz em 1 olhada
- [ ] Keyword no campo (SEO Instagram 2026)
- [ ] Aluno escolheu uma das 2-3 variacoes apresentadas
- [ ] Confronto aplicado se nome muito comum (credencial diferenciadora forcada)

---

## Item 2 — Bio

- [ ] Tipo identificado (GROWTH/SALES/DM/LOCAL)
- [ ] Formula adaptativa do tipo aplicada
- [ ] CRIATIVA (nao cliche — qualquer um do nicho NAO podia ter escrito o mesmo)
- [ ] CLARA (entende em 5s)
- [ ] POSICIONADA (carrega elementos do nucleo: dor/solucao/beneficio/crenca)
- [ ] CTA final ACIMA do link
- [ ] CTA e ESPECIFICO (anuncia o que vem em seguida — nao "link na bio")
- [ ] ≤ 150 caracteres
- [ ] 1 acao primaria (nao buffet de 3+ CTAs)
- [ ] Emojis (se usados) tem FUNCAO, nao decoracao
- [ ] Bluntness > poetry — bio bruta vence

---

## Item 3 — Link da Bio

- [ ] Opcao escolhida (Linktree / LP direta / Pagina sob medida)
- [ ] PRIMEIRO LINK = funil principal (REGRA INEGOCIAVEL)
- [ ] Maximo 4-5 links (decision fatigue)
- [ ] Cada link com LABEL CLARA (nao "clique aqui")
- [ ] Hierarquia visual: hero destacado dos secundarios
- [ ] Estrutura adaptada ao objetivo (sales/growth/dm/local)
- [ ] Se aluno nao tem ativos: PLACEHOLDER + instrucao acionavel registrada

---

## Item 4 — Destaque Sobre Mim

- [ ] Sequencia de 4-6 stories estruturada
- [ ] Cada story com funcao clara (hook / desenvolvimento / CTA)
- [ ] Carrega Pontos 1, 3, 5, 11 + 1 crenca do nucleo
- [ ] Capa-conceito definida: frase + elemento visual + hierarquia
- [ ] Capa COMUNICA o tema em 1s ("bater o olho e saber")
- [ ] NAO contem design (sem paleta/fonte/layout especificado)
- [ ] Oferta de producao apresentada
- [ ] Stories produzidos (se aluno aceitou) — pra cada: texto + nota visual + funcao

---

## Item 5 — Destaque Produto

### Se tem produto:
- [ ] Sequencia de 4-7 stories estruturada
- [ ] Cada story coberta: capa, publico, transformacao, mecanismo, prova, oferta, CTA
- [ ] Carrega Pontos 5, 6, 10, 11 do nucleo
- [ ] Capa-conceito definida
- [ ] Stories produzidos (se aluno aceitou)

### Se NAO tem produto:
- [ ] Placeholder com ESTRUTURA preenchivel
- [ ] Capa-conceito generica recomendada
- [ ] Acao acionavel registrada: "Lancar produto → voltar aqui"

---

## Item 6 — Destaque Depoimentos

### Se tem depoimentos (3+):
- [ ] Capa "DEPOIMENTOS" / "PROVAS"
- [ ] Variedade de formatos planejada (texto, video, print, antes/depois)
- [ ] Ordem por IMPACTO (mais forte primeiro)
- [ ] Cada depoimento com nome + foto (regra)
- [ ] Capa-conceito definida

### Se NAO tem (ou tem <3):
- [ ] Placeholder com instrucao acionavel
- [ ] Lista de COMO coletar (print WhatsApp, video curto, screenshot)
- [ ] Template pra pedir depoimento (se aluno aceitou)
- [ ] Capa-conceito generica recomendada

---

## Item 7 — Pinned 1 (Sobre)

- [ ] Formato definido (recomendado: Reel; alternativa: Carrossel)
- [ ] HOOK forte no primeiro frame (apresentacao magnetica OU pergunta provocativa)
- [ ] Carrega Pontos 1, 3, 5, 11 + crenca central + apresentacao magnetica
- [ ] CTA claro no final ("me segue pra X" / "manda Y no DM")
- [ ] Capa-conceito CRITICAMENTE clara — faz visitante CLICAR
- [ ] Capa-conceito: face + nome OU palavra-titulo
- [ ] Copy produzida (se aluno aceitou) — frame a frame com nota visual
- [ ] Duracao estimada: 30-60s se Reel; 6-10 slides se Carrossel

---

## Item 8 — Pinned 2 (Tese/Crenca)

- [ ] Formato definido (recomendado: Carrossel)
- [ ] CRENCA escolhida da lista do nucleo (a MAIS FORTE)
- [ ] HOOK em formato de gancho (provocacao/curiosidade)
- [ ] Desenvolvimento da tese ao longo dos slides
- [ ] Carrega Ponto 8 (mentira) + Ponto 9 (nao faz sentido) + Crenca
- [ ] CTA + chamada de tribo no final ("se ta comigo nessa, me segue")
- [ ] Capa-conceito CRITICA — a FRASE-BANDEIRA em destaque
- [ ] Capa funciona em texto grande, fundo contrastante
- [ ] Copy produzida (se aluno aceitou) — slide a slide

---

## Item 9 — Pinned 3 (Oferta)

### ESTRUTURA OBRIGATORIA:
- [ ] **RUMINACAO** (slides 1-3): ativa a dor sensorialmente
  - [ ] Usa Ponto 5 (dor) + Ponto 6 (inimigo)
  - [ ] Lista tentativas frustradas (Ponto 7)
  - [ ] Faz SENTIR, nao so reconhecer
- [ ] **SOLUCAO** (slides 4-5): apresenta angulo unico
  - [ ] Usa Ponto 10 (solucao)
  - [ ] Nega solucoes alternativas
  - [ ] Cria desejo PELA solucao antes do produto
- [ ] **PRODUTO** (slides 6-7): apresenta o que vende
  - [ ] Nome + 1 frase do que e
  - [ ] EXATOS 3 beneficios (Ponto 11)
  - [ ] 1 prova rapida
- [ ] **CTA** (slide 8): chamada pra compra
  - [ ] Especifico (nao "saiba mais")
  - [ ] 1 acao primaria
  - [ ] Urgencia OU restricao OU clareza

### Outros criterios:
- [ ] Formato definido (recomendado: Carrossel de 8 slides)
- [ ] Capa-conceito CRITICA — nome do produto + 1 beneficio em ≤5 palavras
- [ ] NAO pulou ruminacao
- [ ] NAO pulou solucao

### Se NAO tem produto:
- [ ] PLACEHOLDER com estrutura preenchivel
- [ ] Lista acionavel registrada

---

## Quality Gate Final (QG-PD-003)

Antes de passar pro chief compilar:

- [ ] 9 itens com check explicito do aluno (display + bio + link + 3 destaques + 3 pinned)
- [ ] Placeholders OK pra itens sem ativo, MAS instrucao acionavel registrada
- [ ] Pelo menos 1 versao de produce-copy chamada (bio formatada OU stories OU copy de pinned)
- [ ] Capas-conceito de TODOS os itens visiveis (destaques + pinned)
- [ ] Lista de pendencias acionaveis consolidada
