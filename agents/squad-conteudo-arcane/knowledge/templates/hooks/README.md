## Como Adicionar

Cria arquivo `.md` com nome descritivo:

```
templates/hooks/
├── meus-hooks-validados.md         # hooks que viraram nos teus posts
├── templates-do-meu-nicho.md       # estruturas comuns no nicho
├── adaptacoes-41-audience.md       # variações dos 41 oficiais
└── ...
```

## Formato Sugerido Por Template

```markdown
## Hook: {nome curto}

**Gatilho(s):** {1-3 dos 7}
**Origem:** {viral que inspirou OU criação própria}
**Usado em:** {posts que aplicaram}

**Estrutura:**
"{template com {placeholders}}"

**Exemplo de aplicação:**
"{exemplo aplicado a um tema específico}"

**Performance:** {se já testou, qual foi}
```

## Quando Rico Consulta

Antes de criar hook do zero pra um roteiro, Rico:

1. Verifica `41-templates-audience.md` (estruturas universais)
2. Verifica esta pasta (templates pessoais do expert)
3. Verifica `swipe-files/hooks-virais/` (hooks reais virais coletados)
4. Se nenhum bate: cria próprio aplicando princípios

Quanto mais essa pasta cresce, melhor o output do Rico (mais customizado pro expert).
