# Estrutura Macro do Roteiro

**Aplicação:** todo roteiro de reels/carrossel/long video segue esta estrutura.
**Princípio:** quebrar nessa ordem porque cérebro humano processa nessa sequência.

---

## A Estrutura

```
═══════════════════════════════════════════════════════════════
HOOK (3s)  →  INTRO FORTE  →  CONTEÚDO  →  CTA + POSICIONAMENTO
═══════════════════════════════════════════════════════════════
  ↓              ↓              ↓                  ↓
PARAR        SUSTENTAR      ENTREGAR          AÇÃO +
SCROLL       CURIOSIDADE    VALOR              CRENÇA
              + ABRIR        + LOOPINGS         CONSTRUÍDA
              LOOPING 1
```

Cada peça tem função específica. Pular ou inverter quebra o roteiro.

---

## 1. HOOK (3 primeiros segundos)

**Função:** PARAR o scroll do feed.

**Tempo:** 3 segundos (limite duro — algoritmo decide entregar baseado nesta janela).

**Threshold de sucesso:** >50% das pessoas que viram passam dos 3s.

### Componentes Obrigatórios

**3 camadas alinhadas:**

| Camada | Função | Exemplo |
|--------|--------|---------|
| **VERBAL** | Frase falada | "Adiar demissão é crueldade" |
| **VISUAL** | O que mostra | Pessoa olhando câmera, expressão firme |
| **TEXTUAL** | Texto na tela | "ADIAR = CRUELDADE" |

**1-3 gatilhos combinados (dos 7):**
1. Recompensa
2. Mistério
3. Popularidade
4. Reputação
5. Crença
6. Disrupção
7. Reconhecimento

### Erros Comuns no Hook

❌ **Verbal genérico:** "Hoje eu vou falar sobre demissão"
❌ **Visual fraco:** cenário comum sem elemento que chama
❌ **Sem texto na tela:** perde quem assiste sem som (70%+ dos espectadores)
❌ **1 gatilho fraco:** "Você sabia que..." (Mistério genérico demais)

✅ **Combo forte:** verbal punchudo + visual marcante + texto chamativo + 2 gatilhos

### Exemplos Bons

**Reels viral (Veridiana Cavalieri — 4M views):**
- Verbal: "O Pedro do jogo sabe por quê liderar"
- Visual: criadora em ambiente neutro, expressão analítica
- Textual: "PEDRO DO FLAMENGO — LIÇÃO PRA LÍDERES"
- Gatilhos: Popularidade (Pedro) + Reconhecimento (líderes)

**Reels viral (Elias Maman):**
- Verbal: "Esposinha aqui 3 vezes por dia é melhor que Durateston"
- Visual: criador segurando pozinho, sem camisa, shape visível
- Textual: "3X POR DIA = MAIS QUE DURATESTON"
- Gatilhos: Recompensa (shape) + Mistério (que pozinho?)

## 2. INTRO FORTE (5-8 segundos)

**Função:** SUSTENTAR a curiosidade. Abre o 1º looping.

**Tempo:** 5-8 segundos depois do hook.

### O Que Faz a Intro Forte Funcionar

**Princípios:**

1. **Sustenta curiosidade do hook** (não decepciona)
2. **Abre o 1º looping** (promessa do que vem)
3. **Dá contexto suficiente** sem entregar tudo
4. **Pode usar identificação ou prova rápida** pra fisgar

**Estrutura:**

```
{Hook} → {Intro forte}

Intro forte segue uma de 3 formas:

A) IDENTIFICAÇÃO:
   "Voce ja sentiu/viveu/passou por isso?"
   → cérebro: "sim, isso é comigo!"

B) PROMESSA:
   "Vou te mostrar X em Y segundos"
   → cérebro: "vale a pena ficar"

C) PROVA RÁPIDA:
   "{Dado/fato curioso} — e isso significa Z"
   → cérebro: "essa pessoa sabe do que fala"
```

### Exemplos

**Após hook "Adiar demissão é crueldade":**

✅ Intro Forte (Identificação + Promessa):
"Você acha que segurar funcionário ruim é ato humano. Vou te mostrar por que é o contrário."

- Identificação: "você acha" (puxa pra crença comum)
- Promessa: "vou te mostrar" (abre looping)
- Disrupção: "é o contrário" (reforça hook)

---

## 3. CONTEÚDO (corpo)

**Função:** ENTREGAR valor. Manter retenção. Construir raciocínio.

**Tempo:** o grosso do vídeo (60-80% da duração).

### Princípios do Conteúdo

**A) Aplicar 3-5 dos 8 Elementos Notáveis:**
- Informação nova
- Valor prático
- Identificação
- Prova / argumentação poderosa
- Fato curioso
- História magnética
- Contra-intuitivo
- Polêmica

**B) Sequência de Loopings:**
- Fecha um, abre outro
- Mantém curiosidade contínua
- 2-3 loopings em vídeo de 60s

**C) Construção de Raciocínio Crescente:**
- Cada parte sustenta a próxima
- Premissa → Quebra → Virada → Prova → Conclusão
- Pessoa entende A → leva pra B → leva pra C

**D) Variação Emocional:**
- Não monoton — varia entre raiva, alívio, choque, curiosidade
- Mapear emoções ao longo do vídeo

**E) Estrutura Micro Hannah (opcional, quando cabe):**
- Conflito → Jeito Errado → Virada → Jeito Certo → Mudança

### Anatomia do Corpo (exemplo de 60s)

```
0:05-0:15 (10s) — PROVA / DADO
"Pesquisa Harvard 2019: 89% dos demissores que adiaram disseram
que deveriam ter agido antes."

Elemento: Prova
Emoção: choque/incredulidade
Função: fecha looping 1 (hook prometeu mostrar — entrega prova)
Abre looping 2: "mas como afeta meu time?"

0:15-0:25 (10s) — IDENTIFICAÇÃO + DADO
"Gallup 2023: 68% dos teus top performers consideram sair
quando convivem 6+ meses com underperformer. Você está perdendo
os bons em silêncio."

Elementos: Prova + Identificação
Emoção: desconforto
Função: aprofunda a tese, identifica situação cotidiana

0:25-0:40 (15s) — HISTÓRIA MAGNÉTICA
"Conheço CEO que adiou demissão de C-level por 2 anos. Resultado:
4 gerentes A-player saíram nesse período. Custo de reposição
estimado: 3 milhões."

Elemento: História magnética + Prova (caso real)
Emoção: choque concreto (números)
Função: fecha looping 2, abre looping 3 ("como demitir sem destruir?")

0:40-0:50 (10s) — VALOR PRÁTICO
"Demissão bem-feita tem 3 elementos:
1. Clareza — pessoa sabe exatamente por que
2. Dignidade — sem espetáculo
3. Velocidade — não arrasta meses"

Elemento: Valor prático
Emoção: alívio/orientação
Função: fecha looping 3, entrega solução
```

## 4. CTA + POSICIONAMENTO (fechamento)

**Função:** CONVERTER atenção em ação + construir CRENÇA.

**Tempo:** 10-15s no fim do vídeo.

### CTA (Call to Action) Contextual

**Regras:**
- DEPOIS da entrega top (não no começo)
- Específico de **reconhecimento** ("compartilhe com [tipo de pessoa]")
- NUNCA genérico ("curte/comenta/segue")
- Conectado ao conteúdo

**Exemplos bons:**

✅ "Compartilhe esse vídeo com todo CEO que está adiando uma demissão difícil — eles precisam ouvir isso."

✅ "Mostra esse vídeo pra todo pai que tem filho na adolescência."

✅ "Manda pra teu cônjuge se você acabou de viver essa situação."

**Exemplos ruins:**

❌ "Curte, comenta, segue, ativa as notificações!"
❌ "Compartilha aí!"
❌ "Me segue pra mais conteúdo!"

### Posicionamento / Crença Construída

**Função:** amarra com a MORAL DA HISTÓRIA. Conecta ao produto/visão de mundo.

**Estrutura:**

```
[Crença forte / visão de mundo] + [Conexão sutil ao produto/expertise]
```

**Exemplos:**

```
"Líder humano não é líder permissivo. Demitir cedo, com clareza,
é ato de liderança. Pensa nisso na próxima vez que tiver essa
decisão na mesa."

→ Crença: "Humano ≠ permissivo"
→ Visão de mundo: clareza > conforto
→ Conexão produto sutil: "decisão na mesa" implica que você
   ajuda a tomar essas decisões (mentoria de gestão)
```

```
"Pais que cedem ao filho criam fracos. Você não tá errando por
ser carinhoso. Tá errando por confundir carinho com permissão.
Pensa nisso essa noite."

→ Crença: "Permissão ≠ amor"
→ Visão: limites são amor
→ Conexão sutil: "pensa nisso" implica reflexão guiada (produto
   de parentagem)
```

## Conexão Entre Os 4 Componentes

```
HOOK (parar) ──┐
                ├─→ Abre LOOPING 1
INTRO (sustentar) ──┘
                    ↓
                    
CONTEÚDO (entregar) ──→ Fecha L1, abre L2, fecha L2, abre L3...
                       ↓
                       Variação emocional + Elementos notáveis
                       ↓
                       
CTA (ação) ──┐
              ├─→ Conversão + Posicionamento
POSICIONAMENTO (crença) ──┘
```

Cada parte serve à próxima. Roteiro coerente flui.

## Tempos Recomendados

### Reels 30-60s (mais comum)

| Componente | Tempo |
|------------|-------|
| Hook | 3s |
| Intro | 5-8s |
| Conteúdo | 30-40s |
| CTA + Posicionamento | 10-12s |

### Reels 60-90s

| Componente | Tempo |
|------------|-------|
| Hook | 3s |
| Intro | 5-8s |
| Conteúdo | 50-70s |
| CTA + Posicionamento | 10-15s |

### Carrossel (10 slides)

| Componente | Slides |
|------------|--------|
| Hook | 1 (capa) |
| Intro/reforço | 1-2 (slide 2 reforça hook) |
| Conteúdo | 5-7 (slides intermediários) |
| CTA + Posicionamento | 1-2 (últimos slides) |

## Resumo da Estrutura

> HOOK (parar) → INTRO FORTE (sustentar + abrir looping) → CONTEÚDO (entregar + loopings + elementos notáveis + raciocínio crescente + variação emocional) → CTA específico + POSICIONAMENTO/CRENÇA construída

Esta é a fórmula. Não tem atalho.
