# Escalar Conteúdo: Formato vs Assunto

**Aplicação:** quando Aria identifica post viral, sugere 2 caminhos de escala.
**Princípio:** acertou padrão? Repete. Mas tem 2 jeitos de repetir.

---

## Os 2 Caminhos (Elias Maman)

```
ACERTOU UM POST VIRAL
        ↓
┌───────┴───────┐
│               │
ESCALAR         ESCALAR
FORMATO         ASSUNTO
│               │
│               │
Mesmo HOOK +    Mesmo TEMA +
outros ASSUNTOS outros HOOKS
```

---

## Caminho 1 — Escalar FORMATO

**O que é:** pegar a ESTRUTURA do hook/formato campeão e aplicar em outros assuntos.

**Funciona porque:** a estrutura já foi validada. Trocar o tema mantém a "mágica".

### Exemplo Concreto

**Post viral original:**
> "Isso é pão. Isso é pão. Isso também é pão. E isso aqui, você acha que é pão?"
>
> (Formato: comparativo visual de variações + revelação)
> (Resultado: 700k views, ganhou 200k seguidores em 1 mês)

**Escalando o formato:**

```
"Isso é OVO. Isso é OVO. Isso também é OVO. E isso aqui?"
"Isso é BATATA. Isso é BATATA. Isso também é..."
"Isso é ARROZ. Isso é ARROZ. Isso também é..."
"Isso é CARNE..."
"Isso é LEITE..."
```

**50+ alimentos comuns, mesmo formato. Cada um vira post.**

### Como Identificar Formato Escalável

Aria pergunta:

1. **A estrutura é replicável?** (pode trocar tema sem perder identidade)
2. **O gancho funciona por estrutura ou por tema específico?**
3. **Existe um "elemento template" claro?** (frase, sequência, padrão visual)

Se sim aos 3 → escalável.

### Quando Escalar Formato

✓ Hook tem estrutura clara repetível
✓ Visual/padrão estabelecido
✓ Performance acima da média em formato específico

✗ Hook depende totalmente do tema (não da estrutura)
✗ Visual irrepetível (evento único)

---

## Caminho 2 — Escalar ASSUNTO

**O que é:** pegar o TEMA validado e fazer outros ângulos/hooks.

**Funciona porque:** público já demonstrou interesse no tema. Outros ângulos puxam mesma audiência.

### Exemplo Concreto

**Post viral original:**
> Tema: "Demitir funcionário bom"
> Hook: "Adiar demissão é crueldade"
> (Resultado: 850k views)

**Escalando o assunto (mesmo tema, hooks diferentes):**

```
1. "3 sinais que você deveria ter demitido há meses"
   (Hook: lista + reconhecimento)

2. "O CEO que adiou demissão por 2 anos perdeu 3 milhões"
   (Hook: caso real + número chocante)

3. "Demitir vs deixar pedir demissão — quem ganha?"
   (Hook: comparativo + pergunta)

4. "Eu adiei demissão de C-level. Ela pediu pra sair. Foi pior."
   (Hook: confissão pessoal)

5. "A demissão mais difícil da minha carreira me ensinou X"
   (Hook: personalização)
```

**5 posts sobre o mesmo tema, hooks diferentes. Cada um pode viralizar.**

### Como Identificar Assunto Escalável

Aria pergunta:

1. **O tema tem profundidade?** (suporta múltiplos ângulos sem ficar repetitivo)
2. **Demanda do público se sustenta?** (comentários pediram aprofundamento)
3. **Existem mais virais do nicho no mesmo tema?** (validação de demanda)

Se sim aos 3 → escalável.

### Quando Escalar Assunto

✓ Tema tem profundidade real (muitos ângulos possíveis)
✓ Comentários do post viral pediram mais
✓ Outros criadores do nicho exploram (validação)
✓ Performance acima da média no tema

✗ Tema esgotou em 1 post
✗ Sem ângulos novos
✗ Saturação do mercado

---

## Comparação dos 2 Caminhos

| Aspecto | Escalar FORMATO | Escalar ASSUNTO |
|---------|-----------------|-----------------|
| Estabilidade | Alta (estrutura validada) | Média (depende de novos ângulos) |
| Variedade pro público | Baixa (parece similar) | Alta (cada post diferente) |
| Risco de saturar | Alto (público cansa do mesmo) | Médio |
| Esforço criativo | Baixo (estrutura pronta) | Alto (cria novos hooks) |
| Crescimento de audiência | Cresce com mesmos seguidores | Atrai mesmos + novos |

**Recomendação prática:** alternar entre os 2 ao longo do tempo.

- 1ª semana: escala formato (rotina rápida)
- 2ª semana: escala assunto (variar)
- 3ª semana: testa formato novo (lapidação 80/20)

---

## Combinação dos 2

Pode escalar AMBOS simultaneamente:

```
Tema: demissão
Formato campeão: "comparativo visual" (isso é X, isso é X...)

Combina:
"Isso é DEMISSÃO BOA. Isso é DEMISSÃO RUIM. E isso aqui?"

→ Aplica formato campeão em tema campeão
→ Probabilidade alta de viral
```

## Cuidados Ao Escalar

### Cuidado 1 — Saturação

Postar 5x seguidas o mesmo formato OU o mesmo tema cansa audiência.

**Solução:** intercalar. 1 do formato campeão, 1 de outro tema, volta no formato.

### Cuidado 2 — Cópia Externa

Você ESCALA o que VOCÊ acertou. Não escala o que outro acertou.

✗ Outro criador viralizou com formato X. Você copia.
- Vira "mais um", sem identidade.

✓ Você acertou formato X. Repete VOCÊ.

### Cuidado 3 — Mudança Sutil

Mesmo formato, mas variar:
- Cenário levemente
- Cores
- Texto na tela
- Bordão (se houver)

Evita virar "o mesmo vídeo".

### Cuidado 4 — Quando PARAR

Sinal de saturação:
- Métricas caindo em posts seguintes do mesmo padrão
- Comentários reclamando ("já vi isso", "tá repetitivo")

Quando ocorre: pausa, vai pra outro formato/assunto. Volta depois.

## Aplicação no Squad

Aria sugere os 2 caminhos no relatório:

```markdown
## Recomendações de Escala

### Caminho A — Escalar FORMATO
Hook estrutura: "{padrão identificado}"
Aplicar em outros assuntos:
1. {ideia 1}
2. {ideia 2}
3. {ideia 3}

### Caminho B — Escalar ASSUNTO
Tema: "{tema validado}"
Outros ângulos:
1. {ideia 1}
2. {ideia 2}
3. {ideia 3}

Recomendação: {qual caminho priorizar e por quê}
```

Expert decide qual seguir (Aria sugere, não executa).
