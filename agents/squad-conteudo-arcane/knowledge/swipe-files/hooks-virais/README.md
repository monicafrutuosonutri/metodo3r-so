# Swipe File — Hooks Virais Coletados

**Status:** vazia por design.

**Quem popula:** o expert ao longo do uso do squad.

**O que entra aqui:**
- Hooks de vídeos virais (do nicho ou de outros)
- Capturados manualmente durante a pesquisa
- Estudados pra extrair padrões (engenharia reversa)

## Diferença Entre `templates/hooks/` e `swipe-files/hooks-virais/`

- **templates/hooks/** — estruturas adaptadas/criadas pelo expert
- **swipe-files/hooks-virais/** — hooks REAIS de virais (com fonte e métricas)

## Como Adicionar

```
swipe-files/hooks-virais/
├── 2026-05-coletados.md            # batch por período
├── nicho-lideranca.md              # por nicho
├── top-100-virais.md               # melhores coletados
└── ...
```

## Formato Por Entrada

```markdown
## Viral: {título curto}

**Hook completo:** "{frase verbal exata do vídeo}"
**Visual:** "{descrição do que mostra na tela}"
**Textual:** "{texto que aparece na tela}"

**Criador:** {nome / @handle}
**Plataforma:** {Instagram / TikTok / YouTube}
**Data:** {YYYY-MM-DD}
**Alcance:** {views aproximadas}
**Engagement:** {se tem dados}

**Gatilhos identificados:** {quais dos 7}
**Por que funcionou:** {análise breve}

**Aplicabilidade pro nicho do expert:**
- Tema: {pode adaptar pra qual tema do pool?}
- Como adaptar: {como aplicaria}

**Link/Screenshot:** {se tem}
```

## Quando Usar

Rico consulta antes de criar hook:
- Procura por nicho similar
- Identifica padrões recorrentes
- Adapta com voz do expert

Sage usa pra engenharia reversa:
- Analisa por que o hook funcionou
- Extrai estrutura
- Aplica na teoria do post

## Quanto Mais, Melhor

Pool de 50+ hooks virais coletados = repertório enorme.
- Permite escolha rápida
- Identifica padrões emergentes
- Antecipa tendências

## Princípios

**Coletar com regularidade:** 5-10 hooks por semana mantém pool fresco.

**Foco em VIRAIS:** só registra hooks que demonstraram performance.

**Variedade:** equilibra gatilhos (não só Mistério, não só Disrupção).

**Análise > Cópia:** Rico extrai ESTRUTURA, não copia texto.
