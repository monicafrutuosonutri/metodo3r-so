# Swipe File — Conteúdos Virais Completos

**Status:** vazia por design.

**Quem popula:** o expert ao longo do uso do squad.

**O que entra aqui:**
- Roteiros completos de vídeos virais (transcritos)
- Carrosseis virais (slides estudados)
- Análise de padrões de estrutura

## Como Adicionar

```
swipe-files/conteudos-virais/
├── reels-virais-nicho.md           # roteiros transcritos
├── carrosseis-top.md               # análise de carrosseis
├── series-episodicas.md            # exemplos de séries
└── ...
```

## Formato Por Entrada

```markdown
## Viral Completo: {título}

**Tipo:** Reels / Carrossel / Vídeo Longo
**Criador:** {nome}
**Alcance:** {views}
**Link/Fonte:** {url se possível}

**TRANSCRIÇÃO/ROTEIRO COMPLETO:**

[Texto integral do vídeo OU slide a slide do carrossel]

---

**ANÁLISE ESTRUTURAL:**

HOOK ({tempo}): "{frase}"
  Gatilhos: {quais}

INTRO ({tempo}): "{frase}"
  Função: {qual}

CONTEÚDO:
  - {tempo}: {elemento notável aplicado}
  - {tempo}: {looping aberto/fechado}
  - ...

CTA ({tempo}): "{frase}"
  Tipo: {específico de reconhecimento / outro}

FECHAMENTO: "{frase}"
  Posicionamento: {qual}

---

**PADRÕES IDENTIFICADOS:**
- {padrão 1}
- {padrão 2}
- {padrão 3}

**APLICABILIDADE PRO EXPERT:**
- Estrutura adaptável a: {temas similares}
- Pode replicar formato em: {ideias específicas}
```

## Quando Usar

**Sage (engenharia reversa):**
- Lê 5-15 virais relevantes durante pesquisa externa
- Extrai padrões pra teoria do post

**Rico (referência estrutural):**
- Pega estrutura validada
- Adapta pra tema do expert
- Sem copiar texto, apenas estrutura

**Iris (validação de tema):**
- Confirma se tema do pool tem precedente viral
- Mostra alcance esperado baseado em similares

## Diferença Pra Outros Swipes

- **templates/hooks/** = estruturas DE HOOK
- **swipe-files/hooks-virais/** = HOOKS REAIS com fonte
- **swipe-files/conteudos-virais/** = CONTEÚDOS COMPLETOS transcritos

## Cadência de Adição

**Recomendado:** 1-3 virais por semana, transcritos e analisados.

Após 6 meses = repertório de 40-100 virais analisados. Vira ouro.

## Princípios

**Transcreve por inteiro:** não anota só "começa com X". Anota TUDO que foi dito.

**Análise > Texto:** o valor tá na ESTRUTURA identificada.

**Diversidade:** vários nichos, vários formatos, vários gatilhos.

**Próprio nicho prioritário:** 60-70% do swipe file deve ser do nicho do expert.
